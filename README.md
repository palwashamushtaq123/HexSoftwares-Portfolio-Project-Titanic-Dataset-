**PORTFOLIO PROJECT:TITANIC SURVIVAL PREDICTION:**
 
 **Project Overview:** :This project is an end-to-end Machine Learning pipeline built on the Titanic dataset. The goal is to analyze passenger data and predict survival outcomes using classification models.

**Workflow:**

Data Cleaning & Preprocessing

Exploratory Data Analysis (EDA)

Feature Engineering

Data Visualization

Model Building

Model Evaluation

**Key Insights from EDA:**

Gender played a major role in survival (females had higher survival rates)
Passenger class strongly influenced survival chances (higher class = higher survival)
Younger passengers had slightly better survival probability
Passengers traveling with family had better survival outcomes

**Feature Engineering:**

New features were created to improve model performance:

FamilySize → Total family members onboard
IsAlone → Whether passenger was traveling alone

**Machine Learning Models:**

Logistic Regression

Decision Tree Classifier

 **Model Performance:**

Logistic Regression → 80% Accuracy

Decision Tree → 77% Accuracy

 **Logistic Regression performed better due to better generalization on unseen data.**
